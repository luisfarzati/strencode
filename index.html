strencode

About: Unicode | UTF-8 | UTF-16 | Base64 | MD5 | SHA-1 | SHA-256 | SHA-512 | SHA-3 | RIPEMD-160

Type any string and it will be converted/encoded/hashed/encrypted automatically using the different encodings and algorithms below. Use this tool for educational purposes, programming, debugging or just for fun!

Written by @luisfarzati as a React learning exercise. Check out the source project at GitHub. This tool uses CryptoJS.
